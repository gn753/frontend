## Commit Convention

| commit type | commit message                            |
| ----------- | ----------------------------------------- |
| feat        | 기능 개발                                 |
| fix         | 버그 수정                                 |
| style       | 불필요한 코드 제거 등                     |
| design      | 디자인 관련 코드                          |
| chore       | 빌드 수정, 패키지 매니저 등 기타 변경사항 |
| move        | 파일이나 폴더 위치 변경                   |
| remove      | 파일이나 폴더 제거                        |
| refactor    | 코드 리팩토링                             |
